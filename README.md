软的转专业难度一直都比较低，去年转专业明显没有前几年火爆，软给了35个名额，实际报名人数就24或25个（具体我忘了）最后录取名单是23人，我大一绩点是3.0，名单上是倒数第四个，我朋友3.3左右的排倒数第八。我面试的时候问的问题（数学怎么考这么差，为什么要转来软院，转来软院之后有什么打算），一共5分钟左右，没怎么为难，后面还简单聊了会儿天。估计是只要绩点不算太差，面试的时候没有发现品行不端正就问题不大。
然后就是你说的准备简历和项目的问题，当时转专业比较匆忙我都没有准备，老师对这些也没有明确要求，但是建议你在开头的自我介绍中简要说明一下（喜欢数据结构和计算机相关的知识），共享屏幕简单看一下github账号里的小项目，这些都是加分项。简历应该不是必须的，具体看今年转专业群里的要求。总之你要在面试中表现出想要转到软院的强烈意愿，但也不要贬低原专业，要让老师觉得你是真的不适合不喜欢原专业，并且能够学下去软件工程，加之差不多的绩点，录取就没问题。
